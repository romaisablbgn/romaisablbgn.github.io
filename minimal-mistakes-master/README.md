# My Personal Website

**Live site:** https://romaisablbgn.github.io

This repository contains the source code for my personal website and blog where I document my journey learning data science and the tools used in the field, including Python, R, SQL, and other data tools.

I am a Management Accounting graduate and now 8-5 employee learning data science at 24. This site serves as a space to share notes, experiments, and reflections as I explore the field.

The website is built with **Jekyll** and hosted using **GitHub Pages**.

---

## Theme

This website uses the **Minimal Mistakes Jekyll theme** created by **Michael Rose**.

Minimal Mistakes is a flexible two column Jekyll theme designed for personal websites, blogs, and portfolios. The design is intentionally minimal so it can be easily customized.

Theme documentation:  
https://mmistakes.github.io/minimal-mistakes/docs/

Original repository:  
https://github.com/mmistakes/minimal-mistakes

---

## Customizations

This site includes several customizations to adapt the theme to my personal brand and blog:

- Site configuration through `_config.yml`
- Custom typography using **Google Fonts**
- Brand fonts: **Instrument Serif** and **Mulish**
- Layout and styling adjustments for a personal blog

More changes may be added as the site evolves.
